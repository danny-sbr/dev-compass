# Print.js

Print.js 是一個輕巧但功能強大的 JavaScript 函式庫，專門用於處理網頁列印需求。這個工具讓網頁列印變得更加簡單且直覺。

## 特色

- 輕量級設計
- 易於使用
- 支援多種列印需求
- 完整的文件支援

## 安裝方式

### 使用 npm 安裝

```bash
npm install print-js --save
```

### 使用 yarn 安裝

```bash
yarn add print-js
```

### 在專案中匯入

```javascript
import printJS from 'print-js'
```
